# DocInfo Engine

![Version](https://img.shields.io/badge/version-1.3.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Photoshop](https://img.shields.io/badge/Photoshop-CC2020+-ff00ff)

Add document specifications as text layer in Adobe Photoshop.

## Features

- Display document dimensions
- Show resolution (PPI, PPCM)
- Color mode and bit depth
- Megapixel calculation
- Aspect ratio (Farey format)

## Installation

Copy to Photoshop Scripts folder

## Usage

File → Scripts → Browse → Select script
