Professional document metadata embedding script for Adobe Photoshop. Automatically creates text layers with technical specifications and document information.

## Features

- Automatic document dimensions extraction
- Resolution and color mode detection
- Creation of metadata text layers
- Customizable text formatting
- Batch processing capability
- Non-destructive operation

## Installation

Copy `DocInfo Engine.jsx` to your Photoshop Scripts folder:
- macOS: `~/Library/Application Support/Adobe/Adobe Photoshop [VERSION]/Presets/Scripts/`
- Windows: `C:\Program Files\Adobe\Adobe Photoshop [VERSION]\Presets\Scripts\`

## Usage

1. Open Photoshop with your project document
2. File → Scripts → Browse → Select `DocInfo Engine.jsx`
3. Script automatically analyzes document
4. New text layer created with document specifications
5. Customize text layer as needed

## Requirements

- Adobe Photoshop CC 2020 or newer
- Active document
